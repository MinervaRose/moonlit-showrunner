# Project Description

Moonlit Showrunner is an AI-assisted short-drama pipeline that turns a simple magical premise into a complete production package and assembled MP4.

The prototype represents a small creative production team as a structured workflow: a showrunner interprets the premise, a scriptwriter builds a six-scene short drama, a storyboarder converts scenes into shots, a visual director creates style and prompt guidance, a continuity checker reviews the emotional arc and consistency, and an editor produces an edit decision list.

The final layer uses Python to assemble title cards, scene cards, captions, and transitions into a short MP4. This keeps the system reproducible and inspectable while leaving clear extension points for image generation, video generation, voiceover, and music.

## Features

- Premise-to-story-package generation
- Six-scene short-drama structure
- Character cards
- Storyboard and camera directions
- Visual style bible
- Image/video prompts
- Continuity report
- Edit decision list
- MP4 export
- Streamlit interface
- Mock mode for testing without API calls
